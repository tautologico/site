IGED – Interpretador Gráfico de Estruturas de Dados

Introdução

No ensino das Estruturas de Dados e seus algoritmos, um dos problemas enfrentados é o grau de abstração necessário para entender as operações com as estruturas na memória. Livros didáticos e outros materiais educacionais sobre o assunto frequentemente usam figuras para ilustrar as estruturas de dados e sua organização, tornando a discussão mais concreta para os alunos. Nos livros, essas figuras são estáticas, apresentando apenas alguns momentos selecionados na execução dos algoritmos que manipulam as estruturas. Existem ferramentas computacionais que mostram animações das estruturas, mas essas animações são normalmente fixas e preparadas com antecedência pelo professor.

O IGED – Interpretador Gráfico de Estruturas de Dados – é uma proposta de ambiente de programação interativo para o ensino de Estruturas de Dados. Neste ambiente o aluno poderá escrever programas para manipular estruturas de dados e visualizar, com animações, o efeito da execução do programa na estrutura de dados utilizada, ajudando na apreensão dos conceitos. O professor poderá criar animações facilmente (apenas escrevendo os programas necessários) e também poderá definir exercícios de programação que deverão ser resolvidos pelos alunos. O ambiente IGED corrigirá automaticamente o resultado dos exercícios de programação, comparando a estrutura de dados obtida pelo programa do aluno com a estrutura desejada (determinada pelo professor).

Estado Atual e Objetivos

Estruturalmente, o IGED inclui um tradutor (compilador), uma máquina virtual, uma interface gráfica (que deve conter editor de programas, janela de visualização das estruturas e vários controles) e os componentes internos relacionados com a avaliação de exercícios.

Atualmente, o projeto conta com alunos de graduação trabalhando no desenvolvimento dos vários componentes. Para um trabalho de mestrado, além do desenvolvimento da ferramenta, deve haver um enfoque de pesquisa na descrição do que há de inovador na abordagem, e na validação da ideia, além das possibilidades educacionais do IGED para o ensino da computação.
